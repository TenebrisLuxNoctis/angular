Présence

!bot Teste la présence du bot

Renvoie un message si le bot est présent

Résumé

!resume Permet d'afficher le résumé du jeu actuel

Affiche l'état des compteurs pour le jeu actuel

Gestion du jeu actuel

!setcurgame GAME Change le jeu actuel pour GAME !curgame Affiche le jeu actuel

Incrémentation d'un critère

!+ critere Ajoute 1 au critère !+ critère valeur Ajoute valeur au critère

Décrémentation d'un critère

!- critere Enlève 1 au critère !- critère valeur Enlève valeur au critère

Mode silencieux

!mute Permet d'activer/désactiver le mode silencieux

Le mode silencieux permet de ne plus afficher de réponse lors de la mise à jour d'un critère (une confirmation est toutefois envoyée en message privé)

Gestion d'un critère

Création / Modification

Lors de la création d'un critère, dans le champ "Résumé" n'entrez pas le caractère "§", il sert à insérer l'état du compteur et ne dois être utilisé que dans le champ "Affichage". Le caractère "#" sert à positionner des "S" en cas de pluriel dans les deux chaînes

Suppression

La suppression d'un critère est immédiate et effective pour tous les jeux concernés.

Gestion d'un jeu

Création / Modification

Pour bénéficier de l'image du jeu, il faut utiliser le même nom que sur twitch. Le nom abrégé n'a aucune incidence et est propre au bot. Il faut sélectionner au moins un critère pour que l'opération soit valide.

Suppression

La suppression d'un jeu est immédiate. Veillez à ne pas supprimer le jeu courant du bot, sans quoi ce dernier pourrait se perdre dans le néant et tout bêtement planter...
